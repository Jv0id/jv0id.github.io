### 24小时内

```
select * from deptdevicepart where department_no=? and device_no=? AND insert_date >= (NOW() - INTERVAL 24 HOUR)
```

### 三天前零点

```
SELECT DATE_FORMAT(DATE_ADD(NOW(),INTERVAL - 3 DAY),'%Y-%m-%d 00:00:00');
```

### 当天零点

```
SELECT DATE_FORMAT(NOW(), '%Y-%m-%d 00:00:00')
```
